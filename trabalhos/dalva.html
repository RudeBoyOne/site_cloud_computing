Serviços da AWS

Armazenamento, Segurança e Bancos de Dados em Cloud

🗄️ Banco de Dados

Amazon Aurora

Banco relacional compatível com MySQL e PostgreSQL, focado em alta performance.

Vantagens
  • Alta performance
  • Escalabilidade automática
  • Alta disponibilidade
Desvantagens
  • Custo elevado
  • Dependência da AWS

Amazon DocumentDB

Banco NoSQL compatível com MongoDB para armazenamento de documentos JSON.

Vantagens
  • Escalável
  • Totalmente gerenciado
  • Compatível com MongoDB
Desvantagens
  • Menos recursos que MongoDB nativo
  • Custo elevado

Amazon DynamoDB

Banco NoSQL rápido e altamente escalável para aplicações modernas.

Vantagens
  • Altíssima velocidade
  • Escala automática
  • Alta disponibilidade
Desvantagens
  • Modelagem complexa
  • Pode gerar custos altos

🔐 Segurança

AWS Backup

Serviço que automatiza e centraliza backups na nuvem.

Vantagens
  • Backups automáticos
  • Gerenciamento centralizado
  • Segurança dos dados
Desvantagens
  • Custo adicional
  • Configuração inicial

AWS IAM

Gerenciamento de usuários e permissões de acesso.

Vantagens
  • Controle de acesso
  • Alta segurança
  • Integração com serviços AWS
Desvantagens
  • Configuração complexa
  • Curva de aprendizado

AWS Shield

Proteção contra ataques DDoS em aplicações na nuvem.

Vantagens
  • Proteção automática
  • Alta disponibilidade
  • Integração com AWS
Desvantagens
  • Versão avançada é cara
  • Uso restrito à AWS

💾 Armazenamento

Amazon EBS

Armazenamento em bloco utilizado com instâncias EC2.

Vantagens
  • Alta performance
  • Snapshots de backup
  • Integração com EC2
Desvantagens
  • Uso limitado ao EC2
  • Custo dependendo do uso

Amazon EFS

Sistema de arquivos compartilhado para múltiplas instâncias.

Vantagens
  • Escalável
  • Compartilhado
  • Fácil de usar
Desvantagens
  • Mais caro que EBS
  • Latência maior

Amazon S3

Serviço de armazenamento de objetos altamente durável.

Vantagens
  • Alta durabilidade
  • Escalável
  • Baixo custo inicial
Desvantagens
  • Latência maior
  • Custo de transferência
