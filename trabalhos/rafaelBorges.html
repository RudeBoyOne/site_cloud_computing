Cloud Computing - FATEC - 5o Semestre

EC2

Amazon Elastic Compute Cloud

Serviço que permite criar máquinas virtuais na nuvem para hospedar aplicações e servidores com escalabilidade sob demanda.

Computação Máquinas Virtuais

S3

Amazon Simple Storage Service

Serviço de armazenamento de objetos utilizado para guardar arquivos, backups e dados com alta durabilidade.

Storage Backup

RDS

Amazon Relational Database Service

Serviço gerenciado de banco de dados relacional que facilita criação, backup e escalabilidade de bancos SQL.

Banco de Dados SQL

Lambda

AWS Lambda

Executa código automaticamente em resposta a eventos sem precisar gerenciar servidores.

Serverless Automação

VPC

Amazon Virtual Private Cloud

Permite criar redes privadas seguras para organizar recursos como servidores e bancos de dados.

Rede Segurança

DynamoDB

Amazon DynamoDB

Banco de dados NoSQL totalmente gerenciado com alta performance para aplicações modernas.

NoSQL Alta Escala

CloudFront

Amazon CloudFront

Rede CDN que distribui conteúdo e aplicações globalmente com baixa latência.

CDN Performance

IAM

AWS Identity and Access Management

Gerencia usuários, permissões e políticas de acesso aos serviços da AWS.

Segurança Permissões

CloudWatch

Amazon CloudWatch

Ferramenta de monitoramento que coleta logs e métricas da infraestrutura na AWS.

Monitoramento Logs
