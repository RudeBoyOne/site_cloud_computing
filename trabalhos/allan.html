Amazon EC2

Servidores virtuais sob demanda

O EC2 permite criar máquinas virtuais na nuvem para executar aplicações, sites e sistemas. É possível escolher diferentes configurações de hardware e pagar apenas pelo tempo em que o servidor estiver ativo.

Amazon RDS

Banco de dados relacional gerenciado

O RDS facilita o uso de bancos de dados SQL na nuvem, cuidando de tarefas como backup, atualizações e recuperação em caso de falhas, permitindo que o desenvolvedor foque apenas na aplicação.

Amazon S3

Armazenamento de arquivos em nuvem

O S3 é utilizado para guardar arquivos como imagens, vídeos, documentos e backups. Ele oferece alta durabilidade e permite acessar os dados de qualquer lugar através da internet.

Amazon DynamoDB

Banco de dados NoSQL rápido

O DynamoDB é um banco de dados não relacional que oferece respostas muito rápidas e escalabilidade automática, sendo ideal para aplicações que precisam lidar com grande volume de acessos simultâneos.

Amazon CloudFront

Distribuição de conteúdo global

O CloudFront é uma rede de distribuição de conteúdo que entrega arquivos e páginas web a partir de servidores próximos ao usuário, reduzindo o tempo de carregamento e melhorando a performance dos sites.

AWS IAM

Controle de usuários e permissões

O IAM é responsável por gerenciar quem pode acessar os recursos da AWS. Com ele, é possível criar usuários, grupos e permissões específicas para manter o ambiente seguro e organizado.

AWS Lambda

Execução de código sem servidor

Com o Lambda, é possível rodar funções na nuvem sem precisar configurar ou manter servidores. O código é executado automaticamente quando ocorre um evento, como o envio de um arquivo ou uma requisição HTTP.

Amazon VPC

Rede privada virtual

A VPC permite criar uma rede isolada dentro da AWS, onde você define sub-redes, regras de acesso e comunicação entre serviços, garantindo maior controle e segurança na infraestrutura.

Amazon SNS

Serviço de notificações

O SNS é utilizado para enviar mensagens e alertas para vários usuários ou sistemas ao mesmo tempo, por meio de e-mail, SMS ou outros serviços, sendo muito útil em arquiteturas baseadas em eventos.
