Amazon Analytics

O que faz: É um conjunto de serviços da AWS para coletar, processar e analisar grandes volumes de dados.
Para que serve: Transformar dados em informações úteis através de relatórios, análises e dashboards.
Quando usar: Quando empresas precisam analisar grandes quantidades de dados ou realizar análise de dados em tempo real.

Data Analysis Business Insights

Amazon DocumentDB

O que faz: É um banco de dados NoSQL compatível com MongoDB que armazena informações em documentos JSON.
Para que serve: Armazenar e gerenciar grandes quantidades de dados usados em aplicações modernas.
Quando usar: Em aplicações web ou mobile que precisam de um banco de dados flexível e escalável.

NoSQL Database MongoDB Compatible

Amazon CodeBuild

O que faz: Serviço de integração contínua que compila código, executa testes e gera versões de software automaticamente.
Para que serve: Automatizar o processo de build e testes durante o desenvolvimento de aplicações.
Quando usar: Quando equipes de desenvolvimento precisam automatizar compilação, testes e integração de código.

CI/CD Build Automation

AWS AppSync

O que faz: Serviço da AWS para criação de APIs utilizando GraphQL.
Para que serve: Permitir que aplicações web ou mobile acessem dados de diferentes serviços de forma rápida e eficiente.
Quando usar: Quando aplicações precisam acessar dados em tempo real ou integrar diferentes serviços usando GraphQL.

GraphQL API Data Integration

AWS IoT Core

O que faz: Plataforma que permite conectar dispositivos de Internet das Coisas (IoT) à nuvem da AWS.
Para que serve: Permitir que dispositivos enviem dados para a nuvem e recebam comandos de controle.
Quando usar: Em projetos de sensores, automação residencial, monitoramento industrial e dispositivos inteligentes.

Connected Devices IoT Communication

Amazon S3

O que faz: Serviço de armazenamento de objetos na nuvem da AWS.
Para que serve: Armazenar arquivos como imagens, vídeos, documentos e backups com alta segurança e durabilidade.
Quando usar: Quando é necessário armazenar arquivos de aplicações, backups ou grandes volumes de dados.

Object Storage Scalable Storage

AWS Lambda

O que faz: Serviço de computação serverless que executa código automaticamente sem precisar gerenciar servidores.
Para que serve:Processar eventos e executar funções quando algo acontece no sistema.
Quando usar: Para automações, processamento de arquivos, APIs serverless ou tarefas que precisam escalar automaticamente.

Serverless Event Processing

Amazon Detective

O que faz: Serviço que ajuda a investigar atividades suspeitas e incidentes de segurança na AWS.
Para que serve:Para que serve: Analisar logs e eventos para identificar possíveis ameaças ou comportamentos anormais.
Quando usar: Quando empresas precisam monitorar segurança e investigar incidentes em ambientes AWS.

Security Analysis Threat Investigation

Amazon CloudFront

O que faz: Serviço de CDN que distribui conteúdo através de servidores localizados em várias partes do mundo.
Para que serve: Acelerar a entrega de sites, APIs, vídeos e arquivos para usuários.
Quando usar: Quando aplicações precisam entregar conteúdo rapidamente para usuários em diferentes regiões do mundo.

CDN Global Delivery
