Serviços da AWS

Explorar tecnologias em nuvem

Por Vitor Tavares Chaves

📡

AWS IoT Core

Serviço da Amazon voltado para soluções de IoT (Internet of Things). Permite criar, monitorar e comunicar-se com 'coisas' através de protocolos como MQTT, integrando camadas de IoT a sistemas de nuvem com sensores e atuadores.

⚡

AWS Lambda

Serviço para rodar código sem administrar servidores. Utilizado para escalar aplicações como APIs, com pagamento apenas pela computação utilizada. Gerencia infraestrutura e segurança entre camadas do sistema.

💾

DynamoDB

Banco de Dados como Serviço (DBaaS) - NoSQL orientado a chave-valor e documento. Utilizado para sistemas web, mobile, IoT e outros. AWS fornece toda infraestrutura com opções de pagamento sob demanda ou provisionados.

📈

Auto Scaling

Serviço de escalamento automático de capacidades em nuvem (como EC2) baseado em média histórica de uso. Monitora continuamente e adiciona/remove CPU conforme necessário, ou em momentos pré-determinados.

⚖️

Load Balancing

Distribui o tráfego de entrada de aplicações entre instâncias EC2 em uma ou mais AZs (Zonas de Disponibilidade). No caso de falha, o tráfego é distribuído entre as instâncias restantes.

💿

AWS Backup

Automatização de tarefas de backup service-by-service de acordo com políticas e cronogramas definidos. Fornece proteção centralizada de dados em toda a infraestrutura AWS.

🔐

Site-to-site VPN

Cria conexão segura entre centro de dados ou rede local com Amazon VPC ou recursos AWS pela internet pública. Possibilita comunicação criptografada e confiável entre ambientes.

🔑

AWS IAM

Identity and Access Management - Gerencia acessos aos recursos AWS. Permite criar usuários, grupos, funções e aplicar políticas JSON. Recomenda-se seguir "menor privilégio" e "confiança zero".

💾

Amazon EBS

Elastic Block Store - Permite criar blocos de armazenamento e anexá-los a instâncias EC2. Utilizado para backup, recuperação de desastres ou armazenamento tradicional em cargas de trabalho na nuvem.
